# 'AI 연산 프레임 전용 스트리밍' 방식 분석

현재는 드론이 보내는 **모든 30프레임**을 백엔드가 받아서 앱으로 즉시 쏴주고 있으며, 그중 6장당 1장만 AI가 개입해 박스를 그립니다.

질문해 주신 아이디어는 **"AI가 연산(검사)을 수행한 바로 그 1장의 핵심 사진 프레임들만 모아서, 앱으로 전송하여 끊김 없는 영상처럼 보여주자"**는 혁신적인 접근법입니다. 
이 방식을 적용했을 때의 기술적 장단점과 실무적 영향을 분석해 드립니다.

---

## 1. 작동 원리 (스트리밍 파이프라인 변화)
- **현재 방식 (Pass-through):** 
  - 백엔드 → 앱: 초당 60장 전송 (60 FPS 영상)
  - UI 체감: 드론 비디오 원본처럼 아주 부드러움. 단, 빨간 박스가 살짝 늦게 따라감.
- **제안 방식 (AI-Filtered):**
  - 백엔드 → 앱: AI가 연산한 프레임(6장 중 1장)만 골라서 전송. 즉, **초당 10장 전송 (10 FPS 영상)**
  - UI 체감: CCTV를 보는 듯한 뚝뚝 끊기는(10 FPS) 느낌이 들지만, 빨간 박스와 실제 크랙의 위치가 **1mm의 오차도 완벽히 맞는 상태로 고정**됨.

---

## 2. 장단점 분석

### 👍 장점 (Pros)
1. **네트워크 대역폭(데이터 소모량) 83% 절감**
   - 1초에 60장을 보내던 것을 10장만 앱으로 쏘게 되므로, 통신 데이터 낭비가 압도적으로 줄어듭니다. 현장에 나가 있는 스마트폰의 데이터 요금을 아끼고 배터리 광탈을 막는 데 огромный 도움이 됩니다.
2. **"완벽한 일치(Zero Visual Delay)" 보장**
   - 시각적으로 빨간 박스가 뒤늦게 쫓아가는 현상 자체가 아예 삭제됩니다. 앱에서 보는 모든 사진은 AI가 직접 들여다보고 그린 "진짜 정답지"이기 때문입니다.
3. **프론트엔드 렌더링 부하 감소**
   - 플러터(Flutter) 앱 프론트엔드도 1초에 60장을 화면에 그리려다 보면 저사양 폰에서는 발열이 생깁니다. 1초에 10장만 렌더링하면 앱이 훨씬 가볍고 부드럽게 돌아갑니다.

### 👎 단점 (Cons)
1. **관리자의 시각적 피로도 (저프레임 체감)**
   - 60FPS의 부드러운 드론 조종 화면을 보다가, 갑자기 10FPS(뚝뚝 끊기는 느낌) 화면을 보게 되면 관리자 입장에서 실시간 비행 상황을 파악하거나 드론을 조종하는 데 멀미나 답답함을 느낄 수 있습니다. (드론 FPV용으로는 부적합해짐)
2. **놓치는 화면 발생**
   - AI가 보지 않고 버리는 5장의 프레임(0.08초 분량) 사이에 드론이 지나쳐버린 찰나의 장면이 있다면, 앱 화면에서는 그 장면이 영원히 노출되지 않고 사라지게 됩니다.

## 3. 구현 난이도 및 방법
구현 자체는 백엔드 `stream_manager.py`에서 단 **두 줄만 수정**하면 되는 극도로 간단한 작업입니다.

```python
# 수정 전 (모든 프레임을 인코딩하여 앱으로 전송 대기)
ret, buffer = cv2.imencode('.jpg', final_output, ...)
self.current_frame = buffer.tobytes()

# 수정 후 (AI 연산을 한 타이밍에만 인코딩하여 전송 대기)
if frame_count % frame_interval == 0:
    ret, buffer = cv2.imencode('.jpg', final_output, ...)
    self.current_frame = buffer.tobytes()
```

## 4. 총평 및 추천
질문하신 방법은 '데이터 통신 최적화'와 '객체 탐지의 시각적 정확성'을 위해 실제로 산업 현장의 AI 드론(송전탑 점검 등)에서 널리 쓰이는 표준 기술 중 하나입니다.

- **만약 사용자(관리자)가 앱을 보면서 직접 드론을 수동 조종(조이스틱)해야 한다면:** 현재의 부드러운 60FPS 방식(Pass-through)을 유지해야 드론이 벽에 부딪히는 것을 막을 수 있습니다.
- **만약 드론이 자동 비행(Auto-pilot) 중이고, 관리자는 그저 크랙이 발견되는 화면 결과만 모니터링하는 목적이라면:** 제안해 주신 **AI 10 프레임 전용 스트리밍 방식**이 배터리/네트워크/발열 모든 면에서 훨씬 더 완벽하고 세련된 구조입니다.
